# Shopping 页面本次重构核对记录

## 1. 商品品类扩充依据
本次商品池从原有纪念品清单升级为 20 类购物分类，优先覆盖中文游客高频搜索的药妆、美妆、零食、家电、伴手礼，再补齐文具手账、动漫/IP、母婴、工艺、香氛等“会专门列清单去扫货”的长尾高热品类。日本购物综合攻略里长期高频出现的药妆、零食、电器、伴手礼结构，适合作为 Top 榜单和分类池的主骨架::cite[315].

文具手账、Loft/MUJI/伊东屋系商品、动漫周边和手办属于更偏“兴趣消费”的补充层，因此在本次 20 类中单独拆出“文具手账”“动漫手办”“IP 周边”，避免被传统伴手礼淹没，也更符合中文内容平台里“文具控 / 二次元 / 手账党”单独做清单的浏览习惯::cite[414].

## 2. Top 50 策展逻辑
Top 50 不再由 rating 自动截取，而是按“高频回购 + 出国送礼稳定 + 日本本地购买场景强 + 类目覆盖完整”四条规则人工策展：前排优先放 SK-II、资生堂红腰子、CPB、Anessa、EVE、龙角散、花王蒸汽眼罩、白色恋人、Royce、东京 Banana、象印、松下等跨平台认知度最高的商品，再向下扩展到文具、工艺、箱包、IP 和母婴清单::cite[430].

伴手礼层面额外向北海道与东京站高频款倾斜，例如白色恋人、Royce、生巧、薯条三兄弟、东京 Banana、Press Butter Sand、Tokyo Milk Cheese Factory 等，适合做 Top 50 的中前段“闭眼带不出错”区，也利于页面形成更明确的送人 / 自用双路线::cite[361].

## 3. 商圈地址 / 营业时间来源说明
商圈字段补充时优先采用官方或半官方来源：例如 SHIBUYA109 官方站明确给出了渋谷店的访问入口与营业时间，可直接支撑“涩谷 109”卡片中的地址与时段字段::cite[383].

东京站一番街采用东京站一番街官方英文站与 GO TOKYO 公开页面做交叉确认，用于支撑“东京站一番街 / Character Street / Okashi Land”相关地址与场景描述::cite[408].

京都站伊势丹采用三越伊势丹官方页面的地址与营业时间信息，适合作为京都站周边大型百货点位的标准来源::cite[451].

大阪“心斋桥筋 / 道顿堀”更像开放式商圈而非单一 mall，因此以 JNTO 官方旅游页面为基础整理区域描述、步行关系和逛街特点，再在页面文案里补充“扫货/补货/夜景”型总结表达::cite[333].

冲绳部分使用 DFS Okinawa 公开页面中的那霸 Omoromachi 地址与每日营业时间，便于在页面中形成可点击地图入口::cite[407].

## 4. 图片修复策略
Top 50 重点商品统一采用 `public/images/sv_top_<id>.svg` 的一物一图/一品牌一图海报策略，保证榜单前排图文一致性更稳定；长尾商品则继续复用更贴切的分类图，命名方式为 `sv_cat_<theme>_<index>.jpg`，例如护肤、美妆、零食、家电、动漫、文具等主题图::cite[315].

这样处理后，Top 50 榜单可以保证强识别度，分类区则通过主题图轮转控制素材成本；如果后续补到更真实的品牌图，只需要替换 `public/images/` 下同名资源，不必改动数据结构或 UI 组件::cite[431].

## 5. 小红书风文案整合方式
本次并未逐句搬运某一篇攻略，而是把中文购物攻略里反复出现的“闭眼买、送同事不出错、补货率高、东京站最后冲、药妆店常年断货款”这类高频表达，转写成更像小红书博主总结口吻的短句，服务于商品 short/why/highlights 与商圈 desc 字段::cite[430].
